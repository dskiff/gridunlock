### Project setup

Requires:

- [git-lfs](https://git-lfs.github.com/)

### Generating icons

1. Update icon in `/src/assets/grid-unlock.svg`
2. Make sure you have version 7+ of [ImageMagick](https://imagemagick.org) installed
3. Run `npm run generate-icons`
